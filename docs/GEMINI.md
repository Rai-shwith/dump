## What is Dump?

A serverless online clipboard service. Users create clipboards at custom URLs,
share text between devices, and optionally protect them with passwords.
No accounts. No email. Anonymous usage.

Live domain: dump.ashwithrai.me
Tech: React + Vite (Cloudflare Pages) + Cloudflare Workers + Cloudflare KV

## Key Decisions Summary

Full details in docs/DECISIONS.md. Quick reference only:

- KV keys: clip:<code>:meta and clip:<code>:content per clipboard
- Global starred list: app:starred (max 5 codes, JSON array)
- Password hashing: SHA-256(password + pepper) via SubtleCrypto
- Owner token: crypto.randomUUID() generated by worker, SHA-256 hash stored in KV
- Owner token stored in browser: localStorage key ownerTokens.<code>
- Owner token bypasses all password prompts
- Clipboard modes: public, reserved, protected
- Expiration: stored as UTC ISO 8601. Frontend converts to local timezone for display
- One-time view: deleted after content is returned, not after password verification
- Starring: public clipboards only, one pin per clipboard, global list max 5
- Reserved keywords blocked: admin, api, raw, json, create, edit, delete, settings, help, docs, host
